Met width: 100%; vult het plaatje de parent over de breedte:

Met width: 200px is de breedte altijd 200 pixels:

Met display: block ;margin-left: auto; en margin-right: auto; komt het plaatje in het midden van de div

main {
    width: 50%;
    margin: 10%;
}
.vullen {
    width: 100%;
}
.klein {
    width: 200px;
}
.midden {
    display: block;
    margin-left: auto;
    margin-right: auto;
    width: 400px;
}
